# Project-For-SpeedOfLight

Этот проект является решением задания на позицию Backend разработчика
Решение данной задачи решено на Java . Хотя можно было на любом другом языке.
Важное условие при решении задания:

- Соблюдение принципов ООП и SOLID
- Так же показать насколько крут твой проект в отличие от других

## Как запускать

У вас должен быть установлен Maven.

`git clone https://github.com/ArcxExe/Project-For-SpeedOfLight.git` - Клонирование проекта
`cd Project-For-SpeedOfLight/ && mvn clean package && java -jar target/solid-fridge-1.0-SNAPSHOT.jar` - Запуск
